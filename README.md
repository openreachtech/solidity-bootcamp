#  Solidity Bootcamp
Crypto Learning ClubのSolidity学習用の教材

## 学習パス（予定）
1. Remixを使いながらSolidityの文法を理解
2. Truffleを使いながら実践的なコントラクトを理解(Uniswap、NFT Marketplace)
3. テストネットへのDeployを通じて、ブロックチェーン特有の概念を理解
4. ベストプラックティスを理解
5. ハッキングに挑戦（Openzeppelinの教材）
